publishImages
=============

Provides a list of :ref:`publishimage-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``publishImages`` are:

	* ``publishImages``: the list of :ref:`publishimage-object` objects
	* ``total`` (long): the total number of objects
