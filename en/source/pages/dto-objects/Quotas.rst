quotas
======

Provides a list of :ref:`quota-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``quotas`` are:

	* ``quotas``: the list of :ref:`quota-object` objects
	* ``total`` (long): the total number of objects
