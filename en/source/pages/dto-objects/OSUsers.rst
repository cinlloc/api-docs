osUsers
=======

Provides a list of :ref:`osuser-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``osUsers`` are:

	* ``osUsers``: the list of :ref:`osuser-object` objects
	* ``total`` (long): the total number of objects
