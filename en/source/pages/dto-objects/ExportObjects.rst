exportObjects
=============

Provides a list of :ref:`exportobject-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``exportObjects`` are:

	* ``exportObjects``: the list of :ref:`exportobject-object` objects
	* ``total`` (long): the total number of objects
