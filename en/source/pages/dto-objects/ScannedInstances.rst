scannedInstances
================

Provides a list of :ref:`scannedinstance-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``scannedInstances`` are:

	* ``scannedInstances``: the list of :ref:`scannedinstance-object` objects
	* ``total`` (long): the total number of objects
