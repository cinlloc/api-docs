partitions
==========

Provides a list of :ref:`partition-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``partitions`` are:

	* ``partitions``: the list of :ref:`partition-object` objects
	* ``total`` (long): the total number of objects
