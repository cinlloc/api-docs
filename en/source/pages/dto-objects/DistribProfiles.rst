distribProfiles
===============

Provides a list of :ref:`distribprofile-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``distribProfiles`` are:

	* ``distribProfiles``: the list of :ref:`distribprofile-object` objects
	* ``total`` (long): the total number of objects
