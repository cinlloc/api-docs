appliances
==========

Provides a list of :ref:`appliance-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``appliances`` are:

	* ``appliances``: the list of :ref:`appliance-object` objects
	* ``total`` (long): the total number of objects
