apiKeyPairs
===========

Provides a list of :ref:`apikeypair-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``apiKeyPairs`` are:

	* ``apiKeyPairs``: the list of :ref:`apikeypair-object` objects
	* ``total`` (long): the total number of objects
