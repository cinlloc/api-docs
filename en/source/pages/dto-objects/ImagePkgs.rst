imagePkgs
=========

Provides a list of :ref:`package-object` objects.

Attributes
~~~~~~~~~~

The list of attributes for ``imagePkgs`` are:

	* ``imagePkgs``: the list of :ref:`package-object` objects
	* ``total`` (long): the total number of objects
