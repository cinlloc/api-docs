Appliance Configuration API Resources
=====================================

	* :ref:`applianceBootScript-download`: Downloads a boot script file
	* :ref:`applianceBootScript-downloadFile`: Downloads a boot script file
	* :ref:`applianceBootScript-upload`: Uploads a boot script file
	* :ref:`uforgeStudioPkgLicense-download`: Downloads the license file attached to an UForge Studio package
	* :ref:`uforgeStudioPkgLicense-delete`: Deletes the license file attached to an UForge Studio package
	* :ref:`uforgeStudioPkgLicense-upload`: Uploads a license file for an UForge Studio package
	* :ref:`uforgeStudioPkgLicense-uploadFile`: Uploads a license file for an UForge Studio package
	* :ref:`applianceBootScript-getAll`: Retrieves all the boot scripts for a particular appliance
	* :ref:`applianceBootScript-create`: Adds a new boot script object to an appliance
	* :ref:`applianceBootScript-deleteAll`: Removes all of the boot scripts attached to an appliance
	* :ref:`applianceBootScript-get`: Retrieves a boot script that is attached to an appliance
	* :ref:`applianceBootScript-update`: Updates a boot script that is attached to an appliance
	* :ref:`applianceBootScript-delete`: Removes a boot script attached to an appliance
	* :ref:`uforgeStudioPkg-create`: Creates a new :ref:`oaspkg-object` that stores the meta-data of an UForge Studio package
	* :ref:`uforgeStudioPkg-get`: Retrieves the UForge Studio package meta-data (:ref:`oaspkg-object`) that has been added to a :ref:`appliance-object`
	* :ref:`uforgeStudioPkg-update`: Retrieves the UForge Studio package meta-data that has been added to an appliance
	* :ref:`uforgeStudioPkg-delete`: Deletes the UForge Studio package meta-data and any associated package binary from the appliance
	* :ref:`uforgeStudioPkg-download`: Downloads the package binary attached to an UForge Studio package object
	* :ref:`uforgeStudioPkg-upload`: Uploads a UForge Studio package binary
