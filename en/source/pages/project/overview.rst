Project API Resources
=====================

	* :ref:`project-getAll`: Retrieves all the projects registered for an operating system
	* :ref:`project-get`: Retrieves the information of a project
	* :ref:`project-update`: Updates the information of a project
	* :ref:`project-delete`: Removes a project from the ``Project Catalog``
	* :ref:`projectLicense-download`: Downloads the license of a project
	* :ref:`projectLicense-uploadChunk`: Uploads a license file for a project
	* :ref:`projectLicense-delete`: Remove a license file from a project
	* :ref:`projectLicense-upload`: Uploads a license file for a project
	* :ref:`projectPkgs-update`: Updates the information of a set of package contained in a project
	* :ref:`projectPkg-deleteAll`: Removes all the package binaries from a project
	* :ref:`projectPkg-getAll`: Retrieves all packages and files for a project
	* :ref:`projectPkg-create`: Adds a new package to a project
	* :ref:`projectPkg-update`: Updates the information of a package in a project
	* :ref:`projectPkg-get`: Retrieves the information of a package contained in a project
	* :ref:`projectPkg-delete`: Removes a package binary from a project
	* :ref:`projectPkg-upload`: Uploads a binary file to project package
	* :ref:`projectPkg-downloadFile`: Downloads the project package binary or archive
	* :ref:`projectPkg-download`: Downloads the project package binary or archive
	* :ref:`project-create`: Create a new project in the ``Project Catalog``
